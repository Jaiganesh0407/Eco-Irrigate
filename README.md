
# 🌿 EcoIrrigate

Smart Solar Drip Irrigation System using Streamlit.

## Features
- Soil moisture monitoring
- Solar energy tracking
- Battery storage visualization
- Pump control (simulated)

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Cloud
1. Upload to GitHub
2. Connect to Streamlit Cloud
3. Select app.py
